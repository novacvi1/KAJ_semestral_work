How to use Mindmap App

  

    
Double-click on canvas to create a new node.

    
Double-clicking on node will give you option to change the text of the node.

    
Press onto a node to move it.

    
To connect two nodes right-click on the first one select "Connect Nodes" and then select the other one.

    
Right-click onto a canvas or node for all options.

    
Select node and press Delete or Backspace to delete node.

    
Buttons in navigations can clear the canvas, save canvas or load previously saved canvas.

    
Also, you can drag and drop a saved canvas to load it.
